---------------------------------------------------------------------------
Requirements
---------------------------------------------------------------------------
  * Java 8
  * Third Party libraries in lib folder, see NOTICE.txt
  
---------------------------------------------------------------------------
Getting started
---------------------------------------------------------------------------

To run the demo you'll need at least one Java capable machine (e.g. Windows 
or Linux).  To get the full experience of the demo try three machines.

For information on how to setup the DivConq demo see the Getting Started page:

http://divconq.com/home/getting-started/
